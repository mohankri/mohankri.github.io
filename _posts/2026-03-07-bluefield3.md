BF-3 is first 400Gb/s Data Processing Unit including PCIe 5.0 Switch and BMC.
It supports RDMA/RoCE, Precision Timing, SDN/NFV Acceleration.
Zero Trust Security (Crypto, Zero Trust Acceleration, NVME/TCP Acceleration)

The DPU hosts many accelarator such as
- ARM Accelerator that runs Linux along with DOCA services.
- Datapath Accelerator(DPA) RISC-V based runs RTOS and it's event based architecture. It has 16 cores
and 16 threads each.

- Accelerated Programmable Pipeline: Deep into internal of NIC. (P4 chip). Allow deep inspection to network packet.

- Other Accelerator that are not progammable.
- 400GBs data flow to CPU and GPU.

Just CUDA is for GPU, DOCA is for Smart NIC to interface with different accelerator that runs on ARM
accelerator.

DOCA provides interface/methadology to interact with Datapath Accelerator(DPA). Example you want
to register for link change event

``` c
#include <stdio.h>
#include <stdlib.h>
#include <signal.h>
#include <doca_log.h>
#include <doca_flexio.h>
#include <doca_event.h>

static volatile int keep_running = 1;

// Handle Ctrl+C to exit cleanly
void int_handler(int dummy) {
    keep_running = 0;
}

int main(int argc, char **argv) {
    struct doca_flexio_dev *flex_dev;
    struct doca_flexio_attr dev_attr;
    struct doca_event_queue *eq;
    struct doca_event event;
    int dev_id = 0; // default FlexIO device / NIC

    signal(SIGINT, int_handler); // Ctrl+C handler

    // Initialize DOCA logging
    if (doca_log_init(DOCA_LOG_INFO) != 0) {
        fprintf(stderr, "Failed to initialize DOCA logging\n");
        return -1;
    }

    printf("Opening FlexIO device %d...\n", dev_id);
    if (doca_flexio_dev_get(dev_id, &flex_dev) != 0) {
        fprintf(stderr, "Failed to get FlexIO device %d\n", dev_id);
        return -1;
    }

    if (doca_flexio_dev_attr_get(flex_dev, &dev_attr) != 0) {
        fprintf(stderr, "Failed to get device attributes\n");
        return -1;
    }

    printf("FlexIO device %d max rate: %u Mbps\n", dev_attr.dev_id, dev_attr.max_rate);

    // Create event queue
    if (doca_flexio_event_queue_create(flex_dev, &eq) != 0) {
        fprintf(stderr, "Failed to create event queue\n");
        return -1;
    }

    printf("Monitoring link events... Press Ctrl+C to exit.\n");

    // Event loop
    while (keep_running) {
        // Non-blocking check for event
        if (doca_event_queue_get(eq, &event) == 0) {
            if (event.type == DOCA_EVENT_LINK_CHANGE) {
                if (event.link_up) {
                    printf("[INFO] Port %d link is UP\n", event.port_id);
                } else {
                    printf("[WARN] Port %d link is DOWN\n", event.port_id);
                }
            }
        }
        usleep(500000); // 0.5s sleep to reduce CPU usage
    }

    printf("Shutting down...\n");

    // Clean up
    doca_event_queue_destroy(eq);
    doca_flexio_dev_put(flex_dev);

    return 0;
}
How It Works

doca_flexio_dev_get() → Opens the FlexIO device (NIC/PCIe)

doca_flexio_event_queue_create() → Sets up an event queue for the device

doca_event_queue_get() → Polls the queue for events

DOCA_EVENT_LINK_CHANGE → Triggered when NIC link goes up or down

event.link_up → 1 = up, 0 = down
```
